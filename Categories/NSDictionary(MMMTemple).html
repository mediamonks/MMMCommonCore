  • -mmm_extendedWithDictionary:

    A dictionary built from the receiver by adding values from another dictionary. The other dictionary can be nil. This is to make it more convenient to add stuff to literal dictionaries, such as Auto Layout metrics dictionaries or CoreText attribute dictionaries.
