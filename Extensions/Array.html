  • mmm_forEachPair(block:)

    Iterates through all neighbouring pairs of elements (a[i], a[i + 1]) in a regular order.

  • firstMap(where:)

    Find the first element that can map to a certain type, it’s like doing a .compactMap {}.first without the overhead of mapping all values first.
