extension Sequence
  • unique(by:)

    Elements of this sequence in the same order but with elements having the same identifier (as given by a closure) occurring only once.

    let countries = [
    ("JP", "Japan", "Tokyo"), ("JP", "Japan", "Osaka"),
    ("IT", "Italy", "Milan"), ("IT", "Italy", "Rome")
]
print(countries.unique { $0.0 })
// Prints [("JP", "Japan", "Tokyo"), ("IT", "Italy", "Milan")]
