LanguageMatchingMode

  • exact

    Only return an element where both the “language” and the “region” parts of the identifier match.

  • allowPartiallyMatching

    If no exact match is available, then return an element where at least the “language” part matches, giving preference to identifiers without the “region” part (e.g. ‘de’ is going to match earlier than ‘de-AT’).
